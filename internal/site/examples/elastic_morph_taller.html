Release notes

The server just morphed this card in place.

  • Elastic re-measures on every morph.
  • The host animates from the old height to the new one.
  • The element stays mounted, so no reflow flash.
  • Mid-flight morphs just retarget the latest height.
  • Reduced-motion users get the same end state, no animation.

Every height change rides the theme's easing and duration.
