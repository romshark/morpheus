A plain block container that smoothly animates its height when the natural size of the content changes.

This state carries a much taller payload, so switching to it from the default animates the host up to the larger content height.

  • Each state swap reads the new natural height.
  • The host transitions from the old height to the new one.
  • Mid-flight swaps retarget the latest height instead of finishing the old animation.
  • Resizing the viewport reflows the content; the ResizeObserver fires and the host re-pins.

Switch back to the default to watch the host shrink the same way it grew.
